# Vagas Dev

Um sistema para busca de vagas de trabalho para desenvolvedores.

O sistema busca novas vagas periodicamente em sites de emprego e as armazena em um banco de dados. Os usuários podem acessar o sistema para buscar vagas de acordo com suas preferências por meio da API.

## Funcionalidades

- Consulta de vagas por meio de filtros:
  - Palavra-chave
  - Localização
  - Tipo de vaga (remota, presencial, híbrida)
  - Vagas para pessoas com deficiência (PCD)
  - Fonte da vaga (Gupy, Linkedin, etc.)
- Armazenamento de vagas em um banco de dados para acesso rápido e eficiente.
- Atualização periódica das vagas para garantir que os usuários tenham acesso às oportunidades mais recentes.
- API RESTful para consulta de vagas, permitindo integração com outras aplicações e serviços.
